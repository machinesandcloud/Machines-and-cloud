Where to use them

Use these artifacts in sequence

  • Use the workflow contract before solution design so the scope stays narrow.
  • Use the control matrix before build approval so reviewers can inspect risk treatment.
  • Use the evaluation harness plan before launch so release criteria are explicit.
  • Use the launch checklist before production rollout and every time autonomy expands.
